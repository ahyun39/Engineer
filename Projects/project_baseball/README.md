# ⚾ KBO 리그 타격 데이터 분석 포트폴리오

## 1. 프로젝트 개요

본 프로젝트는 KBO 리그 팀별 시즌 타격 데이터를 수집·분석하여, 특정 시즌(2020년)의 타격 트렌드에 대한 가설을 검증하는 것을 목표로 합니다.
특히 장타력을 나타내는 주요 지표들의 시즌별 변화 추세를 시각적으로 분석하고 인사이트를 도출했습니다.

### 🎯 분석 가설

> 2020년 KBO 리그는 다른 시즌과 비교해 장타(특히 홈런) 비중이 높을 것이다.

<br>

## 2. 데이터 수집 및 전처리

| 구분          | 내용                           |
| :---------- | :--------------------------- |
| **데이터 출처**  | STATIZ (웹 스크래핑)              |
| **수집 대상**   | 1982~2023년 KBO 시즌별 팀 타격 데이터  |
| **수집 방식**   | BeautifulSoup |
| **최종 데이터셋** | `baseball.csv` (343 rows)    |

### 🛠 주요 전처리 단계

* **연도 정제:**
  STATIZ에서 제공하는 축약 연도(예: 97 → 1997, 0 → 2000, 20 → 2020)를
  `year_set` 함수를 통해 정확한 4자리 연도로 변환.
* **컬럼 필터링:**
  분석에 필요한 주요 타격 지표만 추출
  (`이름`, `연도`, `안타`, `2루타`, `3루타`, `홈런`, `장타`, `OPS`)
* **리그 전체 합계 데이터 생성:**
  팀별 데이터 → `groupby('연도').sum()`으로 연도별 리그 전체 타격 데이터로 변환.

<br>

## 3. 분석 및 시각화

### 3.1. 1982~2023 전체 시즌 타격 지표 추이 분석

* **OPS 트렌드**
  시즌별 OPS 변화를 통해 리그 타격 수준의 전반적인 흐름 파악.
* **홈런 트렌드**
  시즌별 홈런 개수 변화를 시각화하여 장타력 변화를 확인.

### 3.2. 가설 검증: 2019~2023 집중 분석

2020년의 특징을 명확히 보기 위해 최근 시즌만 별도로 분석했습니다.

| 분석 지표      | 계산식             | 의미             |
| :--------- | :-------------- | :------------- |
| **홈런 비율**  | 홈런 ÷ 안타         | 전체 안타 중 홈런 비중  |
| **3루타 비율** | 3루타 ÷ 안타        | 전체 안타 중 3루타 비중 |
| **장타 비율**  | (홈런 + 3루타) ÷ 안타 | 장타 비중 파악       |

<br>

## 4. 핵심 분석 결과

| 항목         | 2020년 특징              | 인사이트                      |
| :--------- | :-------------------- | :------------------------ |
| **OPS**    | 최근 시즌 중 가장 높음 (7.572) | 리그의 전반적인 공격 생산성 증가        |
| **홈런 개수**  | 2019년 이후 최고치 (1363개)  | 2020년 장타력 상승의 핵심 요인       |
| **3루타 비율** | 높지 않음 (0.0135)        | 장타 증가의 중심은 **3루타가 아닌 홈런** |

### 💡 결론

2020년 KBO 리그는 OPS와 홈런 개수가 모두 높아 **장타력(특히 홈런)이 크게 상승한 시즌**으로 확인되었습니다.
다만 3루타 비율 증가가 두드러지지 않아, 2020년의 ‘빅볼(Big Ball)’ 현상은 **홈런 중심의 장타 증가**라는 결과를 확인할 수 있습니다.
